Colors & Backgrounds

You can style:


color
background-color
background-image
---
Color Types

Hex:


#ff0000

RGB:


rgb(255, 0, 0)

HSL (great for design):


hsl(0, 100%, 50%)

Opacity (RGBA):


rgba(0, 0, 0, 0.5)
---
Example:

body {
  background-color: #f4f4f4;
  color: hsl(210, 50%, 20%);
}

.card {
  background-color: rgba(0, 0, 0, 0.5);
}
