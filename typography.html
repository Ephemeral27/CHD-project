Typography

Good typography makes your site instantly better.

Important properties:


font-family
font-size
font-weight
line-height
letter-spacing
text-align

Best Practices:

  • Use Google Fonts for modern fonts.
  • REM is better than PX for responsive scaling.

Example:


body {
  font-family: 'Roboto', sans-serif;
  font-size: 1rem;
  line-height: 1.6;
}

h1 {
  font-weight: 700;
  text-align: center;
}
