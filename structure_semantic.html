Document Structure & Semantics

What it is:
The overall skeleton of an HTML page, using meaningful tags.

Why it’s used:

  • Helps browsers and search engines understand your page
  • Improves accessibility

Important tags:

  • <html>, <head>, <body> → basic structure
  • <header> → top section
  • <nav> → navigation menus
  • <main> → main content
  • <section> → content sections
  • <article> → independent content
  • <footer> → page bottom

Example:


<header>
  <h1>My Website</h1>
</header>

<main>
  <section>
    <h2>About Me</h2>
    <p>Hello! I love coding.</p>
  </section>
</main>

<footer>
  <p>© 2026</p>
</footer>
