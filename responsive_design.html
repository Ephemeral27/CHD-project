Responsive Design

Websites must work on phones, tablets, and desktops.

Using Media Queries:

@media (max-width: 768px) {
  body {
    background: red;
  }
}

Explanation:

  • @media → applies styles under certain conditions
  • max-width: 768px → applies styles on smaller screens
---
Use Flexible Units:
  • % → percentage-based sizing
  • rem → relative to root font size
  • vw → viewport width
  • vh → viewport height
Example:

.container {
  width: 80%;
}

h1 {
  font-size: 2rem;
}

.hero {
  height: 100vh;
}
---
