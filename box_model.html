The Box Model

Every HTML element is a box.

It has:

  • Content
  • Padding → inside spacing
  • Border
  • Margin → outside spacing
Example:

div {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}
---

Content-box vs Border-box

Default (content-box):
Width does NOT include padding and border.

Better practice:


box-sizing: border-box;

This makes layout easier and more predictable.

---

Margin Collapse

Vertical margins can combine into one.

This often confuses beginners because:

  • Two vertical margins may merge
  • The larger margin wins
