Attributes & Accessibility

What it is:

  • Attributes give extra info about elements (id, class, src, alt, name, etc.)
  • Accessibility makes your site usable for everyone, including screen readers

Why it’s used:

  • Styling and targeting elements (CSS/JS)
  • Ensuring content is accessible to all users

Examples:


<img src="cat.jpg" alt="A cute gray cat" id="cat-image">

<input type="text" name="username" 
       placeholder="Enter your name" required>

CSS Stuff – Selectors

Basic Selectors:


/* Element selector */
p {
  color: black;
}

/* Class selector */
.card {
  background-color: white;
}

/* ID selector */
#main {
  width: 100%;
}
