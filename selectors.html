Relationship Selectors

Descendant Selector:


div p { }

All <p> elements inside a <div>.

Child Selector:


div > p { }

Only direct children.

Adjacent Sibling Selector:


h1 + p { }

The first <p> after <h1>.

Attribute Selector:


input[type="text"] { }
---

Pseudo-classes

They style elements based on state.


button:hover { }

input:focus { }

li:nth-child(2) { }
---

Pseudo-elements

Create virtual elements for design.


p::before { content: "⭐"; }
