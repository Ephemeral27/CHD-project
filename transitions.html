Transitions (Smooth Animations)

Transitions make changes smooth instead of instant.

Example:

button {
  transition: background 0.3s ease;
}

button:hover {
  background: blue;
}

Explanation:

  • transition → defines what property animates
  • 0.3s → duration of animation
  • ease → animation speed curve
