Text & Content Elements

What it is:
Tags that display text and structure content.

Why it’s used:

  • Organize text meaningfully
  • Make content readable

Common tags:

  • <h1> - <h6> → headings
  • <p> → paragraphs
  • <span> → inline text container
  • <div> → block container for layout
  • <br> → line break
  • <hr> → horizontal line

Example:


<h1>Welcome!</h1>
<p>This is a <span>simple</span> paragraph.</p>
<hr>
