Lists

What it is:
Lists let you display items in an ordered or unordered way.

Types:

  • <ul> → Unordered list (bullets)
  • <ol> → Ordered list (numbers)
  • <dl> → Description list (terms & definitions)

Basic example:


<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
